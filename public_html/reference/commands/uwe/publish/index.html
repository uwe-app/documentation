To publish for the stage environment:

uwe publish stage

Usage

uwe-publish 0.41.0
Publish a website

USAGE:
    uwe publish [FLAGS] <env> [project]

FLAGS:
    -e, --exec       Allow hook command execution
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <env>        Publish environment
    <project>    Project path [default: .]
