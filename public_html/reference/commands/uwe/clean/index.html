To remove the build directory for a project:

uwe clean

Usage

uwe-clean 0.42.22
Remove the build directory

USAGE:
    uwe clean [project]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <project>    Project path [default: .]
