server

HomeDocumentationReferenceCommandsuweserver

The server command serves website files in a directory:

uwe server build/release

Now you can visit the site at http:://localhost:8888.

The server does not list directory content; it is expected that an index.html file exists for each directory.

Usage

uwe-server 0.42.22
Serve static files

USAGE:
    uwe server [FLAGS] [OPTIONS] [--] [target]

FLAGS:
    -e, --exec              Allow hook command execution
        --help              Prints help information
    -i, --include-drafts    Include drafts
    -o, --offline           Offline mode, do not attempt plugin installation
    -O, --open              Launch a web browser
    -s, --skip-build        Do not create a release build for projects
    -V, --version           Prints version information

OPTIONS:
    -h, --host <host>            Bind address for the web server
    -m, --member <member>...     Filter on workspace members
    -p, --port <port>            The port number
        --ssl-cert <ssl-cert>    Path to an SSL certificate file [env: SSL_CERT]
        --ssl-key <ssl-key>      Path to an SSL key file [env: SSL_KEY]
        --ssl-port <ssl-port>    The port number for SSL

ARGS:
    <target>    Project or directory to serve files from [default: .]
