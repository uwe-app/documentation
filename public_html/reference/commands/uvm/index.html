uvm

UWEDocumentationReferenceCommandsuvm

The version manager uvm(1) provides commands for upgrading and managing installed versions of the platform tools; to view help on the command line run uvm help <cmd>.

  • install

    Install a specific version

  • list

    List versions

  • prune

    Remove old versions

  • remove

    Remove a release version

  • uninstall

    Remove the platform tools

  • update

    Upgrade the platform tools

  • use

    Set the current version
