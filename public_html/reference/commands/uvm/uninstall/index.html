uninstall

HomeDocumentationReferenceCommandsuvmuninstall

To completely remove the platform tools run:

uvm uninstall

You must confirm when prompted.

Usage

uvm-uninstall 0.48.6
Uninstall the platform tools

USAGE:
    uvm uninstall

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information
