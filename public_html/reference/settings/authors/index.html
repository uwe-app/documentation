Authors

HomeDocumentationReferenceSettingsAuthors

The [authors] table is used to define the list of authors for a project, see author attribution for more details and examples.

  • name The author name (required)
  • url Link to a website for the author (optional)
  • avatar Link to an image for the author (optional)

The url and avatar must be absolute URLs.
