A common pattern is to show notes such as tips, warnings or info messages. The std::note plugin provides a note block partial which makes it easy to integrate notes in to your pages.

You should add the dependency to your site settings to use it:

[dependencies]
"std::note" = { version = "~1" }

Usage

Use the note partial like this:

{{#> note}}
This is a note with *seamless* Markdown handling.
{{/note}}

Renders to:

This is a note with seamless Markdown handling.

Type

Add a type which is appended to the class:

{{#> note type="warn"}}
This is a warning.
{{/note}}

Renders to:

This is a warning.

Label

Add a label to render a span inside:

{{#> note type="error" label="error"}}
This is an error.
{{/note}}

Renders to:

error

This is an error.

Style

If you want to use the default styles apply them in the dependency:

[dependencies]
"std::note" = { version = "~1", apply = { styles = [ "**" ] } }
